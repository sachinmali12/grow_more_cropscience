Grow More Cropscience Website
================================
https://growmorecropscience.com/

This repository contains the static marketing site for Grow More Cropscience, showcasing agricultural products, company information, and contact details.

## Structure

- `index.html` – Landing page with hero, product highlights, and brand story.
- `all-products.html` and `product_details.html` – Product catalogue and detail layouts that pull data from `products.json`.
- `about-us.html`, `contact-us.html`, `gallery.html` – Supporting pages for company background, inquiries, and media.
- `styles/` – Global stylesheet (`index.css`) applied across pages.
- `assets/` – Product imagery, branding, and supporting media.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in a modern browser, or serve the project with any static file server (for example, `npx serve .`).

## Customization

- Update product information in `products.json`; ensure any referenced images are stored in `assets/`.
- Adjust global styles in `styles/index.css`.
- Replace or add media assets in `assets/` while preserving optimized formats for web delivery.

## Deployment

This site can be hosted on any static hosting provider (e.g., Netlify, GitHub Pages, Vercel, Amazon S3). Upload the repository contents or configure your host to serve the root directory.
